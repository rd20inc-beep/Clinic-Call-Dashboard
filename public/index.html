Setup Instructions

  1. Open chrome://extensions in Google Chrome
  2. Enable "Developer mode" (top right toggle)
  3. Click "Load unpacked" and select the whatsapp-extension folder from this project
  4. Open web.whatsapp.com and scan the QR code
  5. Click the extension icon and set the server URL to this dashboard's URL
  6. The bot will automatically respond to messages and send appointment reminders!
